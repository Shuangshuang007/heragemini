#### 2. **Token泄露风险** ✅ 确认需要修复
**问题**：
- 在多个文档中发现了完整的Bearer Token明文：
  - `CHATGPT_APP_STORE_SUBMISSION_CHECKLIST.md`
  - `MCP_TEST_RESULTS.md`
  - `MCP_INTEGRATION_GUIDE.md`
  - `MCP_IMPLEMENTATION_PLAN.md`
  - `CHATGPT_APPS_INTEGRATION_GUIDE.md`
- Token: `YOUR_MCP_SHARED_SECRET`

**修复方案**：
1. 立即生成新token（至少32字符的随机字符串）
2. 更新Vercel环境变量 `MCP_SHARED_SECRET`
3. 删除所有文档中的token明文，替换为占位符 `YOUR_MCP_SHARED_SECRET`
4. 全仓grep确认不再出现旧token

**验收标准**：
```bash
grep -r "hera_mcp_secret_2025" .  # 应该返回0结果
```

**修复方案**：
- 明确决定：提交11个工具还是裁剪为9个核心工具
